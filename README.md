# WhatsApp Appointment Reminder

A full-stack appointment reminder application built using React, Node.js, Express, and Supabase.

## Features

* Create appointments
* Save appointments in Supabase
* View appointments in a dashboard
* Simulated WhatsApp confirmation messages
* REST API integration

## Tech Stack

* React
* Node.js
* Express.js
* Supabase
* JavaScript

## API Endpoints

### POST /appointments

Creates a new appointment.

### GET /appointments

Returns all saved appointments.

## Future Improvements

* Real WhatsApp integration using Twilio
* Automated reminders using cron jobs
* Authentication and authorization
* Appointment editing and deletion
